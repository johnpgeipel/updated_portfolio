Clicky Game

Description

Clicky Game is a React based memory game.

The user clicks on the shuffling tiles to receive points, however if the same tile is clicked twice, the game is over. If the user can click all the tiles exactly once, they win!

Technologies

HTML, CSS, Bootstrap, Javascript, React
