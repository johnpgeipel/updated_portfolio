Project Details

Application

UpStream Group Project

Description

The concept for this project was to create an app that allows users to search for a movie, then add the movie to a "Watchlist." The user may then view the watchlist and ammend it, whether by clicking "watched" or "delete" buttons.The user may also submit a review of the movie for future reference.


Future Development

We hope to also incorperate GuideBox API data, so the app will also present streaming services available for each selected movie.

Made with HTML, CSS, Bootstrap, JavaScript, JQuery, Express, Node, Handlebars, OMDB API, MySQL & AOS (Animate on Scroll).
